# my-first-flask-app
Reference from Udemy course (rest-api-flask-and-python)

## Set up
### Create virtual env
```buildoutcfg
python -m venv venv
source venv/bin/activate
```
### Install packages
```buildoutcfg
pip install -r requirements.txt


python3.7 app.py
```
## Run the server
```buildoutcfg
FLASK_APP=<main_file> flask run
```
